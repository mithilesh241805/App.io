- 🎁 30 GB signup bonus for all users
- 📺 Ad rewards system (100–500 MB per ad, 2 GB/day cap)

- 🗑️ Recycle Bin with 30-day auto-purge
- 🔗 Secure file sharing with 6-digit encrypted access codes
- ⬇️ Download Manager with quota tracking
- 💳 Razorpay payment integration (UPI QR + cards)
- 📸 Manual UPI screenshot fallback payment
- 🤖 AI storage optimization (Gemini 1.5 Flash)
- 🛡️ Admin panel (user management, revenue, payment approval)
- 📱 Flutter Android app with AdMob rewarded ads
- 🌐 React web app with glassmorphism UI
- 🌙 Dark/Light mode
- 🔄 GitHub Actions CI/CD for all three platforms
- 🐳 Docker Compose for local development and production
